开源项目

web框架及相关

  • playframework: typesafe官方推荐
  • liftweb: 老牌，view-first
  • scalatra: 简洁
  • circumflex: 传统mvc，还提供了ORM
  • unfiltered: 干净
  • Spray 构建于akka这上的轻量级rest库
  • Play2-mini: 构建于playframework之上的轻量级rest库
  • scalate: 基于scala的模板引擎，提供了多种语法，一定有你喜欢的风格

并发

  • Akka: typesafe官方推出的并发库。英文文档，老猪翻译的文档

数据库

  • scalaquery
  • squeryl

序列化

  • salat: Salat is a simple serialization library for case classes.

JSON

  • jerkson: 在强大好用的jackson上包装了一层
  • lift-json

构建工具

  • sbt

测试

  • scalatest
  • specs2

其它

  • parboiled: 简洁的解析器，可以解析自定义的语法和格式。与scala自带的combinator.Parsers类似，但功能更为强大一些。

国产

  • scalaconsole: 可执行scala代码的GUI界面，比纯命令行方便多了
