(2014-09-12) 如何在sbt中定义一个最简单的hello world任务

广告：从今天开始用云梯翻墙

使用taskKey函数，定义一个TaskKey类型的key，然后再给它赋一个函数即可。

build.sbt

name := "hello"

version := "1.0"

scalaVersion := "2.11.0"

lazy val hello = taskKey[Unit]("An example task")

hello := { println("Hello, world") }

运行：

$ sbt hello
Hello, world

注意：

  1. 中间的空行不能少
  2. 通常使用lazy以避免某些初始化顺序问题

show

另外，如果想看hello的返回值，使用show：

$ sbt
> show hello
[info] ()
